## ✨ Особенности и преимущества

Voxentia разработана как высокотехнологичный персональный ассистент, который полностью работает на вашем собственном оборудовании, используя современную модульную архитектуру.

| Функция | Описание |
|---|---|
| 🧠 **Локальная LLM** | Работает на базе [Ollama](https://ollama.com/) — высокая производительность локально (по умолчанию: `phi3`). |
| ⚛️ **Современный React UI** | Профессиональный модульный интерфейс на React, Vite и Tailwind CSS. Дизайн в стиле Glassmorphism. |
| 🎭 **Система ролей** | Переключайтесь между **Профессиональным помощником**, **Дружелюбным собеседником** и **Академическим тьютором**. |
| 🎙️ **Многоязычный голос** | Интегрированные STT (Whisper) и TTS (Silero) для русского, английского и немецкого языков с синхронизацией губ. |
| 🧍 **3D Цифровой аватар** | Рендеринг в реальном времени через [Three.js](https://threejs.org/) с автоматической сменой пола в зависимости от голоса. |
| 📄 **Интеллект документов** | Загрузка PDF для мгновенного RAG-анализа (Retrieval-Augmented Generation) через ChromaDB. |
| 🔌 **Плагинная архитектура** | Модульный бэкенд, позволяющий легко добавлять новые инструменты (календарь, заметки и т.д.). |
| 🔒 **Приватность прежде всего** | Все сервисы работают в Docker-контейнерах. Никакой телеметрии и облачных API. |

## 🏗️ Техническая архитектура

Voxentia использует профессиональную микросервисную архитектуру, координируемую через Docker Compose.

```text
voxentia-assistant/
├── backend/app/                # FastAPI — Основная логика и оркестрация
│   ├── api/                    # Версионированные REST-эндпоинты (v1)
│   ├── core/                   # Настройки Pydantic, база данных и безопасность
│   ├── models/                 # Модели SQLAlchemy для хранения данных
│   ├── schemas/                # Схемы Pydantic для валидации API
│   └── services/               # Бизнес-логика (чат, голос, RAG)
│
├── frontend/                   # React — Модульное веб-приложение
│   ├── src/components/         # Разделенные UI-компоненты (Sidebar, Chat, Avatar)
│   ├── src/hooks/              # Пользовательские хуки React (Audio Manager и др.)
│   └── src/types/              # Общие определения TypeScript
│
├── core/                       # Общий фреймворк Python (система плагинов)
├── tts-server/                 # Движок Silero TTS (Flask Wrapper)
├── whisper-server/             # Движок faster-whisper STT (Flask Wrapper)
├── data/                       # Хранение: векторная БД, история чатов, аудио-кэш
└── assets/                     # Статика: 3D-модели (.glb), текстуры
```

## 🚀 Быстрый старт (Установка)

### Требования
- **Docker Desktop** (с Compose v2+)
- **Windows / Linux / macOS**
- **Ollama**, установленная локально.

### 1. Настройка и запуск (Windows)
Мы предоставляем нативный скрипт управления для Windows:
```powershell
# Установка зависимостей
.\run.bat install

# Запуск всего стека
.\run.bat up
```

### 2. Настройка и запуск (Linux/macOS)
```bash
# Запуск стека Docker
make docker-up
```

Интерфейс доступен по адресу: **`http://localhost:80`** (Nginx) или **`http://localhost:8000`** (Backend Dev).

## 🛠️ Конфигурация

### Переменные окружения (.env)
| Переменная | По умолчанию | Описание |
|---|---|---|
| `DEFAULT_MODEL` | `phi3` | Модель LLM для чата. |
| `DEFAULT_LANGUAGE` | `ru` | Язык по умолчанию (en, de, ru). |
| `OLLAMA_URL` | `http://ollama:11434` | URL к сервису Ollama. |
