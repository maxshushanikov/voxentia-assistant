# 🤖 Voxentia – ИИ-цифровой ассистент

> **Полностью локальная, приватная система ИИ-аватара** с рендерингом 3D в реальном времени, многоязычным голосовым взаимодействием, распознаванием речи и системой знаний на основе документов. Без облачных сервисов. Никакие данные не покидают твой компьютер.

**[🇬🇧 English Version](README.md) | [🇩🇪 Deutsche Version](README_DE.md)**

---

## ✨ Возможности

| Функция | Описание |
|---|---|
| 🧠 **Локальная LLM** | Работает на [Ollama](https://ollama.com/) с `phi3` (заменяемая) — 100% на твоём железе |
| 🎙️ **Локальное распознавание речи** | [faster-whisper](https://github.com/SYSTRAN/faster-whisper) — модель `base`, многоязычная, без Google/Apple |
| 🔊 **Локальный синтез речи** | [Silero TTS](https://github.com/snakers4/silero-models) — русские, английские и немецкие голоса |
| 🧍 **3D-аватар** | Рендеринг в реальном времени через [Three.js](https://threejs.org/) — мужские и женские GLB-модели |
| 💬 **Многоязычный чат** | Полная поддержка EN / DE / RU в интерфейсе, промптах LLM и голосах TTS |
| 📄 **Знания из документов (RAG)** | Загружай PDF — аватар отвечает на вопросы по твоим документам через [ChromaDB](https://www.trychroma.com/) |
| 🎭 **Эмоции аватара** | Синхронизация губ, автономное моргание, анимация ожидания, триггеры эмоций через эмодзи |
| 🔒 **100% конфиденциально** | Все ИИ-сервисы работают в Docker-контейнерах — никаких внешних API-вызовов |

## 🚀 Быстрый старт

### Требования

- **Docker Desktop** (с Docker Compose v2+)
- **~6 ГБ ОЗУ** для всех сервисов
- **~3 ГБ места на диске** для моделей и контейнеров
- Современный браузер с поддержкой WebGL (Chrome, Firefox, Edge)

### 1. Клонировать репозиторий

```bash
git clone <url-репозитория>
cd digital_avatar
```

### 2. Скачать LLM-модель (только в первый раз)

```bash
# Сначала запустить контейнер Ollama
docker compose up ollama -d

# Скачать языковую модель по умолчанию
docker exec -it digital_avatar-ollama-1 ollama pull phi3

# Скачать модель эмбеддингов для RAG (поиск по документам)
docker exec -it digital_avatar-ollama-1 ollama pull nomic-embed-text
```

### 3. Собрать и запустить все сервисы

```bash
docker compose up --build
```

> ⏳ **Первый запуск занимает 5–10 минут** — Docker строит образы, Whisper скачивает модель (~150МБ).

### 4. Открыть в браузере

```
http://localhost:8000
```

## 🎮 Использование Voxentia

### Выбор языка
На стартовом экране выбери **EN**, **DE** или **RU** перед нажатием «Старт». Это установит язык для интерфейса, системного промпта LLM и голоса TTS.

### Панель управления

| Кнопка | Действие |
|---|---|
| 🏠 **На главную** | Вернуться на стартовый экран для смены языка |
| 🎙️ **Говорить** | Нажать для начала записи → нажать снова для остановки и транскрибирования |
| 🔊 **Тест звука** | Проверить вывод аудио |
| 📹 **Видеозвонок** | Переключить режим видеозвонка |
| 📷 **Камера** | Включить веб-камеру |
| 📄 **Загрузить** | Загрузить PDF-документ для обучения аватара |
| Выпадающий список голосов | Выбрать из 4 голосов (Baya, Kseniya, Eugene, Aidar) |
| Выпадающий список эмодзи | Вручную вызвать мимику |

### Голосовой ввод (Whisper)
1. Нажать **🎙️ Говорить** — микрофон активируется
2. Чётко произнести сообщение
3. Нажать **🎙️ Говорить** снова — аудио отправляется в локальный Whisper
4. Транскрипция появляется в чате → аватар отвечает

### Загрузка документов (RAG)
1. Нажать **📄 Загрузить** и выбрать PDF-файл
2. Дождаться подтверждения **✅ OK**
3. Задавать аватару вопросы по содержимому документа
4. Аватар извлечёт нужные фрагменты и ответит на их основе

## ⚙️ Конфигурация

### Переменные окружения

Создать файл `.env` (скопировать из `.env.example`):

```env
# Настройки LLM
DEFAULT_MODEL=phi3
OLLAMA_TIMEOUT=120

# Язык
DEFAULT_LANGUAGE=ru

# URL сервисов (внутренняя сеть Docker)
OLLAMA_URL=http://ollama:11434
TTS_URL=http://tts-server:5002
WHISPER_URL=http://whisper-server:5003
```

### Размер модели Whisper

Отредактировать `docker-compose.yml` для выбора модели Whisper:

```yaml
whisper-server:
  environment:
    - WHISPER_MODEL=base   # Варианты: tiny, base, small, medium, large-v3
```

| Модель | Размер | Скорость | Точность |
|---|---|---|---|
| `tiny` | ~75 МБ | Быстрейшая | Хорошая |
| `base` | ~145 МБ | Быстрая | Лучше ✅ стандарт |
| `small` | ~460 МБ | Средняя | Отличная |
| `medium` | ~1,5 ГБ | Медленная | Превосходная |

### Доступные TTS-голоса

| ID голоса | Язык | Пол |
|---|---|---|
| `baya` | RU/EN/DE | Женский |
| `kseniya` | RU/EN/DE | Женский |
| `eugene` | RU/EN/DE | Мужской |
| `aidar` | RU/EN/DE | Мужской |

## 🛠️ Разработка

### Система модулей фронтенда

Все модули фронтенда используют нативные ES-модули (без бандлера):

```javascript
// Управление состоянием — реактивное хранилище
import { appState } from './modules/core/State.js';
appState.update('language', 'ru');
appState.subscribe('language', (val) => console.log('Язык изменён:', val));

// i18n
import { i18n } from './modules/core/I18n.js';
i18n.setLanguage('ru');
i18n.t('send_btn'); // → 'Отправить'
```

### Добавление нового языка

1. Добавить переводы в `frontend/static/modules/core/I18n.js`
2. Добавить голоса в `frontend/static/modules/core/State.js`
3. Добавить `lang-btn` в `frontend/index.html`
4. Добавить системный промпт в `backend/core/config.py`
5. Добавить код языка в `whisper-server/server.py` → `LANG_MAP`

### Пересборка после изменений кода

```bash
# Пересобрать только бэкенд
docker compose up -d --build avatar-server

# Пересобрать только TTS-сервер
docker compose up -d --build tts-server

# Пересобрать Whisper-сервер
docker compose up -d --build whisper-server

# Полная пересборка
docker compose up --build
```

### Очистка кэша браузера

После изменений фронтенда всегда делай жёсткую перезагрузку:
- **Windows/Linux**: `Ctrl + Shift + R`
- **macOS**: `Cmd + Shift + R`

## 🔍 Диагностика и устранение неполадок

### Просмотр логов сервисов

```bash
docker logs digital_avatar-avatar-server-1
docker logs digital_avatar-tts-server-1
docker logs digital_avatar-whisper-server-1
docker logs digital_avatar-ollama-1
```

### Распространённые проблемы

| Проблема | Причина | Решение |
|---|---|---|
| Нет звука | Аудио браузера заблокировано | Сначала нажать **🔊 Тест звука** |
| Аватар в T-позе | Нет idle-анимации | Проверить консоль — анимации должны быть в `assets/idle/` |
| TTS молчит при EN/DE | Старый контейнер в кэше | `docker compose up -d --build tts-server` |
| Загрузка даёт 405 | Бэкенд не пересобран | `docker compose up -d --build avatar-server` |
| Микрофон показывает ⏳ | Whisper ещё загружается | Проверить `docker logs digital_avatar-whisper-server-1` |
| LLM отвечает на неправильном языке | Старый контекст сессии | Нажать **🏠 На главную** для сброса сессии |

### Логи консоли браузера

Открыть DevTools (`F12`) → Консоль:
- `🚀 VOXENTIA AI SYSTEM v3.0 LOADED` — приложение инициализировано
- `[State Update]` — реактивные изменения состояния
- `📂 Loading N variations...` — загрузка анимаций аватара
- `🎙️ Transcribed [ru]:` — вывод Whisper
- `🔊 [TTS] GENERATING AUDIO` — обработка TTS
