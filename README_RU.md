## ✨ Особенности и возможности

Voxentia спроектирована как высокотехнологичный персональный ассистент, который работает исключительно на вашем оборудовании, используя современную модульную архитектуру.

| Особенность | Описание |
|---|---|
| 🧠 **Локальная LLM** | Работает на базе [Ollama](https://ollama.com/) (по умолчанию: `phi3`) — 100% выполнение на вашем железе. |
| 🎨 **Премиальный UI** | Современный интерфейс в стиле "Claude-meets-Material-You". Стеклянные панели, адаптивная навигационная панель и плавная микро-анимация. |
| 🎭 **Система персон** | Выбор между ролями **Эксперт**, **Дружелюбный компаньон** и **Академический тьютор**, которые меняют поведение ИИ. |
| 🎙️ **Голос без задержек** | Интеграция [faster-whisper](https://github.com/SYSTRAN/faster-whisper) для распознавания речи и [Silero TTS](https://github.com/snakers4/silero-models) для синтеза. |
| 🧍 **3D Цифровой аватар** | Рендеринг в реальном времени через [Three.js](https://threejs.org/) с процедурной анимацией и липсинк-эффектом. |
| 📄 **Интеллект документов** | Загрузка PDF для мгновенного анализа через RAG (Retrieval-Augmented Generation) на базе [ChromaDB](https://www.trychroma.com/). |
| 🔍 **Веб-поиск** | Интегрированный инструмент поиска позволяет ассистенту получать актуальную информацию из сети в реальном времени. |
| 📹 **Мультимодальное зрение** | Интеграция с веб-камерой позволяет ассистенту «видеть» и описывать окружающую обстановку или пользователя. |
| 🔒 **Приватность прежде всего** | Все сервисы работают в Docker-контейнерах. Никакой телеметрии, облачных API или отслеживания данных. |

## 🏗️ Техническая архитектура

Voxentia построена на микросервисной архитектуре, координируемой через Docker Compose.

```
voxentia-assistant/
├── backend/                    # FastAPI — Ядро и оркестрация
│   ├── api/                    # REST-эндпоинты (чат, документы, транскрипция, WebRTC)
│   ├── core/                   # Конфигурация, БД и системные промпты
│   ├── models/                 # Pydantic-схемы для валидации API
│   └── services/
│       ├── llm.py              # Взаимодействие с Ollama (Phi3/Llava) + Tool calling
│       ├── tools.py            # Определения инструментов (поиск, погода)
│       ├── tts.py              # Клиентская логика синтеза речи Silero
│       └── rag.py              # Парсинг документов и векторный поиск (ChromaDB)
│
├── frontend/                   # Vanilla JS — Модульное веб-приложение
│   ├── index.html              # MD3 верстка и точка входа
│   └── static/
│       ├── main.js             # Контроллер и жизненный цикл приложения
│       ├── components/         # UI модули (чат, элементы управления)
│       ├── modules/            # Логика движка (Three.js, Аватар, Аудио)
│       ├── stores/             # Глобальное управление состоянием
│       ├── utils/              # API помощники, WebRTC и форматирование
│       └── styles/             # Модульные CSS стили
│
├── tts-server/                 # Silero TTS Engine (Flask wrapper)
├── whisper-server/             # faster-whisper STT Engine (Flask wrapper)
├── data/                       # Хранилище: Векторная БД, История, Документы
└── assets/                     # Статика: 3D-модели (.glb), Текстуры
```

## 🚀 Быстрый старт (Установка)

### Требования
- **Docker Desktop** (с Compose v2+)
- **NVIDIA GPU** (опционально, для ускорения, но работает и на CPU)
- **Минимум 8ГБ ОЗУ** (рекомендуется 16ГБ)

### 1. Клонирование и настройка
```bash
git clone <your-repo-url>
cd digital_avatar
cp .env.example .env
```

### 2. Загрузка ИИ-моделей
```bash
# Запуск Ollama
docker compose up ollama -d

# Загрузка необходимых моделей
docker exec -it digital_avatar-ollama-1 ollama pull phi3
docker exec -it digital_avatar-ollama-1 ollama pull nomic-embed-text
```

### 3. Запуск системы
```bash
docker compose up --build
```
Интерфейс доступен по адресу: **`http://localhost:8000`**

## 🎮 Руководство по интерфейсу

### Адаптивная навигация
Боковая панель слева обеспечивает быстрый доступ к функциям:
- 🏠 **Главная**: Сброс текущей сессии.
- 📄 **Загрузка**: Прикрепить документы для анализа ИИ.
- 📹 **Видеозвонок**: Активация режима иммерсивного общения.
- 📷 **Камера**: Включение веб-камеры для визуальных задач.
- 🔊 **Тест звука**: Проверка аудиовыхода и активация контекста браузера.

### Боковая панель чата (в стиле Claude)
Премиальная панель для текстового взаимодействия:
- **Выбор голоса**: Доступно 4 уникальных голоса (Baya, Kseniya, Eugene, Aidar).
- **Выбор языка**: Переключение между английским, немецким и русским на лету.
- **Выбор персоны**: Мгновенная смена характера ИИ.
- **Переключатель поиска**: Включение/выключение веб-поиска в реальном времени.

## 🔍 Устранение неполадок

| Проблема | Решение |
|---|---|
| **NotReadableError** (Мик) | Закройте другие приложения, использующие микрофон. Voxentia попробует переподключиться. |
| **Нет звука** | Нажмите кнопку "Тест звука", чтобы активировать аудио-контекст браузера. |
| **Большая задержка** | Убедитесь, что достаточно ОЗУ. Попробуйте модель `tiny` для Whisper в `docker-compose.yml`. |
| **Утечка промпта** | Убедитесь, что используете последнюю версию `llm.py` с обновленными стоп-токенами. |
