# 🤖 Voxentia – ИИ-цифровой ассистент

> **Полностью локальная, приватная система ИИ-аватара** с рендерингом 3D в реальном времени, многоязычным голосовым взаимодействием, распознаванием речи и системой знаний на основе документов. Без облачных сервисов. Никакие данные не покидают твой компьютер.

**[🇬🇧 English Version](README.md) | [🇩🇪 Deutsche Version](README_DE.md)**

---

## ✨ Возможности

| Функция | Описание |
|---|---|
| 🧠 **Локальная LLM** | Работает на [Ollama](https://ollama.com/) с `phi3` — 100% на твоём железе |
| 🎨 **Интерфейс MD3** | Современный дизайн **Material Design 3** с навигационной панелью и профессиональными иконками |
| 🎭 **Система персон** | Динамическое переключение ролей: **Эксперт-консультант**, **Дружелюбный помощник** и **Академический наставник** |
| 🛠️ **Поддержка инструментов** | Архитектура вызова функций (поиск, погода и др.), интегрированная в логику LLM |
| 🎙️ **Локальное STT** | [faster-whisper](https://github.com/SYSTRAN/faster-whisper) — модель `base`, многоязычная |
| 🔊 **Локальный синтез** | [Silero TTS](https://github.com/snakers4/silero-models) — качественные голоса EN / DE / RU |
| 🧍 **3D-аватар** | Рендеринг через [Three.js](https://threejs.org/) с синхронизацией губ и управлением мимикой |
| 📄 **Знания из документов** | Загрузка PDF и поддержка RAG через [ChromaDB](https://www.trychroma.com/) |
| 🔒 **Приватность** | Все ИИ-сервисы работают локально в Docker — данные не покидают компьютер |

## 🚀 Быстрый старт

### Требования

- **Docker Desktop** (с Docker Compose v2+)
- **~6 ГБ ОЗУ** для всех сервисов
- **~3 ГБ места на диске** для моделей и контейнеров
- Современный браузер с поддержкой WebGL (Chrome, Firefox, Edge)

### 1. Клонировать репозиторий

```bash
git clone <url-репозитория>
cd digital_avatar
```

### 2. Скачать модели (только в первый раз)

```bash
# Запустить контейнер Ollama
docker compose up ollama -d

# Скачать языковую модель и модель эмбеддингов
docker exec -it digital_avatar-ollama-1 ollama pull phi3
docker exec -it digital_avatar-ollama-1 ollama pull nomic-embed-text
```

### 3. Запустить все сервисы

```bash
docker compose up --build
```

## 🎮 Использование Voxentia

### Выбор языка и персоны
Через новую панель **Top App Bar** вы можете в любой момент переключить язык и выбрать одну из **Персон** (Эксперт, Помощник, Наставник). Это мгновенно изменит поведение и стиль общения аватара.

### Панель навигации (Sidebar)

| Иконка | Действие |
|---|---|
| 🏠 | **Главная**: Вернуться на стартовый экран |
| 📄 | **Загрузка**: Загрузить PDF для базы знаний RAG |
| 📹 | **Видеозвонок**: Переключить режим видеозвонка |
| 📷 | **Камера**: Активировать веб-камеру |

### Основное управление

*   **🎙️ Микрофон**: Нажать для записи, нажать снова для отправки.
*   **🔊 Динамик**: Провести тест звука.
*   **🎭 Лицо**: Ручное управление мимикой аватара.

## 🔍 Диагностика и устранение неполадок

### Распространённые проблемы

| Проблема | Причина | Решение |
|---|---|---|
| Нет звука | Аудио заблокировано | Сначала нажать на иконку **Динамика** |
| Ошибка микрофона | Hardware lock | Приложение само выполнит «Nuclear Reset» — подождите 2 сек. |
| Аватар в T-позе | Нет анимации | Проверьте `assets/idle/` на наличие GLB-файлов |
| LLM отвечает не так | Старый контекст | Нажмите на иконку **Дома** для сброса сессии |
