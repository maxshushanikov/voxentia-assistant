# Erste Schritte mit Voxentia

Nachdem du Voxentia installiert hast, kannst du direkt mit deinem neuen Assistenten interagieren.

## Das System initialisieren
1. Öffne `http://localhost:8000`.
2. Wähle deine bevorzugte Sprache (Deutsch, Englisch oder Russisch).
3. Klicke auf **System initialisieren**. Der 3D-Avatar wird geladen.

## Den Chat nutzen
Du kannst Nachrichten entweder tippen oder per Spracheingabe (Mikrofon-Symbol) senden.

### Beispiele zum Ausprobieren:
- **Wetter**: "Wie ist das Wetter in Berlin?"
- **Suche**: "Suche nach Informationen über Quantencomputer."
- **Jobs**: "Finde Python Entwickler Jobs in Hamburg."
- **Lernen**: "Erstelle ein Quiz zum Thema Geographie."
- **Kalender**: "Was habe ich heute für Termine?"

## Plugins verwalten
Standardmäßig sind alle installierten Plugins aktiv. Wenn du den Intent eines Plugins ansprichst (z.B. "Job"), öffnet sich das Plugin-Panel automatisch auf der rechten Seite, um dir strukturierte Ergebnisse anzuzeigen.
